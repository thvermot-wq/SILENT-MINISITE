# Silent Injury — mini-site EPK (one-page)

Contenu :
- `index.html` (site statique)
- `assets/presskit.pdf` (press book)
- `assets/logo.png`, `assets/hero.jpg`, `assets/album-cover.jpg`
- `assets/og-cover.jpg` (aperçu lors du partage)
- `assets/gallery/` + `assets/gallery/thumbs/` (photos optimisées)

## Tester en local
Ouvrez `index.html` dans votre navigateur.

## Mettre à jour les liens (2 minutes)
Dans `index.html`, remplacez :
- [URL_DU_SITE] (quand GitHub Pages vous donne l’URL)
- [LIEN_STREAMING] (Spotify/Deezer/Apple si vous en avez)
- [LIEN_YOUTUBE], [LIEN_INSTAGRAM]
- [LIEN_TECH_RIDER] (PDF)
- [LIEN_LIVE_VIDEO] (si dispo)
- [LIEN_DOSSIER_PHOTOS_HD] (Drive/Dropbox, optionnel)

## Publier sur GitHub Pages
1. Créez un repo (ex: `silent-injury-epk`)
2. Uploadez tout le contenu de ce dossier à la racine du repo
3. GitHub → Settings → Pages → Deploy from branch → `main` / root
4. URL : `https://<user>.github.io/<repo>/`

Lien direct press book :
`https://<user>.github.io/<repo>/assets/presskit.pdf`
